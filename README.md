# maazhusain.com

Personal portfolio website for Maaz Husain — Cybersecurity Analyst & Senior Quality Engineer.

🔗 **Live site:** [maazhusain.com](https://maazhusain.com)

## Stack

- HTML, CSS, JavaScript — no frameworks, no dependencies
- Hosted on GitHub Pages

## Structure

```
├── index.html   # Markup and content
├── style.css    # All styles (tokens, layout, components, dark mode)
├── script.js    # Theme toggle, scroll reveal, tweaks panel
└── CNAME        # Custom domain config for GitHub Pages
```

## Features

- Light / dark mode with localStorage persistence
- Scroll-triggered reveal animations
- Responsive layout down to mobile
- Sections: Experience, Projects, Skills, Certifications, Education, Contact
